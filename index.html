Bangkit Academy 2022

Bangkit Academy adalah program persiapan karir yang didukung oleh Google,GoTo, Tokopedia, dan Traveloka. Pada program ini, saya memilih cloud computing learning path. Produk yang dihasilkan dari Kerjasama saya dan tim pada program bangkit academy ini adalah Skinkr. Skinkr adalah aplikasi mobile yang dapat mendeteksi jerawat pada wajah dengan memberikan input berupa foto wajah pengguna. Backend dari aplikasi ini dibuat dengan menggunakan cloud run yang merupakan bagian dari google cloud platform services.
